# Task 1 – HTML Portfolio Website

This project is part of my **Full Stack Web Development (MERN Stack) Internship**.

In this task, I created a **basic portfolio website using HTML** to demonstrate fundamental HTML concepts such as page structure, semantic elements, images, links, and forms.

---

## 📌 Project Overview

The portfolio website includes the following sections:

- Header section
- Navigation menu
- About section
- Portfolio / Projects section
- Contact form
- Footer section

The purpose of this task is to understand **HTML structure, elements, attributes, and semantic tags**.

---

## 🛠 Technologies Used

- HTML5
- Semantic HTML
- Basic form elements
